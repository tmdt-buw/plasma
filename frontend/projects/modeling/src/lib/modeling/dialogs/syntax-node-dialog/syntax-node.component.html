Syntax Node

The ROOT node is the beginning syntax node of your data source. It cannot be mapped

A syntax node is a representation of an element in the original data schema. If it's a leaf node, a semantic class can be mapped to it by dragging. If it's not a leaf node, it indicates a structural element in the data, e.g. an object or an array. Those cannot be mapped but used for orientation.
